# Job Market Salary Analysis Using PostgreSQL:

## Project Overview
This project analyzes job market data to extract salary insights, experience requirements, and highest paying companies using SQL.

## Dataset Information
- Original Dataset: 10,000+ records
- Upload Version: 1,000-record sample
- Columns: 10

## Folder Structure
- dataset/ -> Contains sample dataset
- sql/ -> Contains SQL scripts
- outputs/ -> Contains query results

## Business Questions Answered
- Total numbeer of job listings
- Detection of incomplete records
- Removal of duplication records
- Top 50 senior-level jobs
- Salary range by job title
- Highest paying company per role

## Tools Used
- PostgreSQL
- SQL (Window Functions, Aggregations)
- GitHub
